# [!DNL Audience Manager] [!UICONTROL Device-based Destinations]

Esta página incluye los [!DNL Audience Manager] [destinos basados en dispositivos](/help/using/features/destinations/add-device-based-destinations.md) solo. Entrada [!DNL Audience Manager], [!UICONTROL device-based destinations] también se denominan [!UICONTROL server-to-server destinations]. Para obtener más información sobre una integración, consulte [Intercambio de Adobe](https://exchange.adobe.com/experiencecloud.html) anuncio (si está disponible) o póngase en contacto con su [!DNL Adobe] Consultor o Atención al Cliente.


* **Última actualización** = Muestra la última fecha en la que se produjo una actualización para este [!UICONTROL device-based destination].
* **Actualizaciones** = Indica si [!DNL Audience Manager] envía datos en tiempo real o por lotes (1 día) a este [!UICONTROL destination].
* **Dessegmentar** = La eliminación de la segmentación describe procesos que descalifican y eliminan perfiles de dispositivos de [!UICONTROL segments]. Su capacidad para eliminar un perfil de dispositivo de una [!UICONTROL segment] depende de la opción de dispositivo utilizada para crear una [!UICONTROL Profile Merge Rule]. La eliminación de la segmentación también se utiliza en el contexto de [!DNL GDPR] eliminar solicitudes. Leer [Reglas de combinación de perfiles y procesos de eliminación de la segmentación de dispositivos](/help/using/features/profile-merge-rules/merge-rule-unsegment.md) y [Audience Manager Se Asocia Con Capacidades De Eliminación De La Segmentación](/help/using/overview/data-security-and-privacy/data-privacy-requests.md#aam-partners-with-unsegmentation). [!DNL Audience Manager] envía información de eliminación de la segmentación a [!UICONTROL destinations] marcado con **Sí** más abajo.
* **Recibiendo [!DNL IAB TCF] señales de consentimiento** = El [!DNL Audience Manager Plug-in for IAB TCF] le permite reenviar la [!DNL IAB TC] cadena a socios de activación respetando las opciones de privacidad de los usuarios. Leer [Complemento de Audience Manager para el TCF de IAB](/help/using/overview/data-security-and-privacy/aam-iab-plugin.md).
* **[!DNL ICDS]** = Esta columna enumera los socios que admiten [!UICONTROL Instant Cross-Device Suppression]. Más información [Eliminación instantánea entre dispositivos](/help/using/features/profile-merge-rules/instant-cross-device-suppression.md).
* **ID de dispositivos móviles** = Esta columna enumera los socios que pueden recibir ID de dispositivos móviles de [!DNL Audience Manager].

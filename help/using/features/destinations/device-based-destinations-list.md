# [!DNL Audience Manager] [!UICONTROL Device-based Destinations]

Esta página incluye la [!DNL Audience Manager] [destinos basados en dispositivos](/help/using/features/destinations/add-device-based-destinations.md) solo. En [!DNL Audience Manager], [!UICONTROL device-based destinations] también se denominan [!UICONTROL server-to-server destinations]. Para obtener más información sobre una integración, consulte [Adobe Exchange](https://exchange.adobe.com/experiencecloud.html) listado (si está disponible) o póngase en contacto con su [!DNL Adobe] Consultor o Servicio de atención al cliente.


* **Última actualización** = Muestra la última fecha en la que se actualizó esto [!UICONTROL device-based destination].
* **Actualizaciones** = Indica si [!DNL Audience Manager] envía datos en tiempo real o por lotes (1xday) a [!UICONTROL destination].
* **Desegmentación** = La eliminación de la segmentación describe los procesos que descalifican y eliminan perfiles de dispositivos de [!UICONTROL segments]. Su capacidad para eliminar un perfil de dispositivo de un [!UICONTROL segment] depende de la opción de dispositivo utilizada para crear un [!UICONTROL Profile Merge Rule]. La eliminación de la segmentación también se utiliza en el contexto de [!DNL GDPR] eliminar solicitudes. Lectura [Reglas de combinación de perfiles y procesos de eliminación de la segmentación de dispositivos](/help/using/features/profile-merge-rules/merge-rule-unsegment.md) y [Audience Manager Asociado Con Funciones De Eliminación De La Segmentación](/help/using/overview/data-security-and-privacy/data-privacy-requests.md#aam-partners-with-unsegmentation). [!DNL Audience Manager] envía información sin segmentar a la [!UICONTROL destinations] marcado con **Sí** más abajo.
* **Recepción [!DNL IAB TCF] señales de consentimiento** = La variable [!DNL Audience Manager Plug-in for IAB TCF] le permite reenviar la variable [!DNL IAB TC] cadena a socios de activación respetando las opciones de privacidad de los usuarios. Lectura [Complemento Audience Manager para IAB TCF](/help/using/overview/data-security-and-privacy/aam-iab-plugin.md).
* **[!DNL ICDS]** = Esta columna enumera los socios que admiten [!UICONTROL Instant Cross-Device Suppression]. Más información [Eliminación instantánea entre dispositivos](/help/using/features/profile-merge-rules/instant-cross-device-suppression.md).
* **ID de dispositivos móviles** = Esta columna enumera los socios que pueden recibir ID de dispositivos móviles desde [!DNL Audience Manager].

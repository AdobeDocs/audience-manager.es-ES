# Husos horarios en Audience Manager{#time-zones-in-audience-manager}

Audience Manager utiliza la hora universal coordinada (UTC) en toda su interfaz de usuario.

A menos que se especifique lo contrario, todas las fechas e intervalos de fechas que puede seleccionar en la interfaz de usuario de [!DNL Adobe] [!DNL Audience Manager] son el [Tiempo Universal Coordinado (UTC)](https://www.timeanddate.com/worldclock/timezone/utc). Por ejemplo, en el flujo [Crear destino](../features/destinations/create-cookie-destination.md#segments-mapping), al asignar segmentos a su destino, la fecha de inicio y finalizaci√≥n que seleccione es medianoche (UTC). Lo mismo se aplica a todas las fechas de [!DNL Audience Manager].

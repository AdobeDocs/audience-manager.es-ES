# Husos horarios en Audience Manager{#time-zones-in-audience-manager}

El Audience Manager utiliza la hora universal coordinada (UTC) en toda su interfaz de usuario.

A menos que se especifique lo contrario, todas las fechas e intervalos de fechas que se pueden seleccionar en la interfaz de usuario [!DNL Adobe] [!DNL Audience Manager] son [Hora universal coordinada (UTC)](https://www.timeanddate.com/worldclock/timezone/utc). Lo mismo se aplica a todas las fechas de [!DNL Audience Manager].
